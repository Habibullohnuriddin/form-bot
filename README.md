# Form-Bot Telegram bot

Welcome to Form-Bot Telegram Bot Repository! This bot helps users collect information through a series of steps.

## Overview

Form-bot Telegram bot was created for an educational center that teaches programming technologies. The bot receives the information of the students who want to study in the educational center and sends it to the admin. The main features of the bot: before using the bot, you must fulfill the terms of the bot, that is, you will be a member of 3 telegram channels and click the confirmation button. The bot will test you for your ability. If the verification is successful, the bot will send you a survey.

## Getting Started

### Prerequisites

Programs and libraries that need to be installed to run the bot: Node.js, Telegram API, MongoDB and Telegraph library

### Installation

1. Clone the repository: `git clone https://github.com/your-username/form-bot.git`
2. Install dependencies: `npm install`
3. Run the bot: `nodemon app.js`

## Usage

Enter the bot and press the "Start" button, the bot will ask you for 3 conditions, that is, to become a member of Telegram channels. You cannot use the bot unless your membership is verified, so you must be a member of the channels. Once the membership is approved, the bot will ask you to fill out a survey, and complete that survey using the examples shown in the bot. If the filled information is completed successfully, the bot will send your information to the admin and they will contact you soon.
